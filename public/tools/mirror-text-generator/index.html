Mirror Text Generator - Flip & Reverse Text Online

Our mirror text generator creates horizontally flipped and reversed text using Unicode characters. Type any text and instantly see it mirrored as if reflected in a looking glass. Combined with our upside-down text effect, you get two powerful text flipping tools in one place. Both effects produce pure Unicode text that can be copied and pasted into any text field on any device.

Two Flip Modes

We offer two distinct text flipping effects:

  • Mirror Text (Horizontal Flip) - Reverses the character order and replaces each character with its horizontally mirrored Unicode equivalent. "Hello" becomes something like "ollɘH" where each letter appears as its left-right reflection. This mimics looking at text in a mirror.
  • Upside Down (Vertical Flip) - Reverses the character order and replaces each character with a Unicode character that looks like the original rotated 180 degrees. "Hello" becomes "plɹoM ollǝH". This creates the illusion of text that has been physically turned upside down.

How Mirror Text Works

Mirror text uses a character substitution map that pairs each letter with a Unicode character resembling its horizontal reflection. For example, "b" and "d" are natural mirrors of each other, "p" and "q" are mirrors, and "R" maps to "Я" (the Cyrillic character that looks like a backwards R). The text is also reversed in order so that it reads from right to left, completing the mirror illusion. Not every character has a perfect mirror in Unicode, so some letters (especially symmetric ones like A, H, M, O, T, W, X) remain unchanged since they look the same when mirrored.

How Upside-Down Text Works

The upside-down effect performs two operations: first, each character is replaced with a Unicode character that visually resembles the original letter rotated 180 degrees (for example, "a" becomes "ɐ", "e" becomes "ǝ", "!" becomes "¡"). Then the entire string is reversed so the text reads correctly when physically flipped. Even without rotating your screen, the upside-down text is immediately eye-catching and fun to read.

Punctuation Gets Flipped Too

Our generators do not just flip letters. Common punctuation marks are also mapped to their flipped equivalents: parentheses are swapped ( becomes ), question marks become inverted question marks, brackets and angle brackets are mirrored, and slashes are reversed. This attention to detail makes the flipped text more convincing and complete.
