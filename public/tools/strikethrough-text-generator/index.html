Strikethrough Text Generator - Cross Out Text Online

Our strikethrough text generator adds a horizontal line through each character of your text using Unicode combining characters. The result looks like crossed-out or deleted text, and it can be copied and pasted into any text field on any device. Strikethrough is one of the most popular text effects for social media, used daily by millions to show corrections, add humor, or create dramatic emphasis in posts and bios.

Two Strikethrough Modes

We offer two distinct strikethrough styles, each using a different Unicode combining character:

  • Single Strikethrough (U+0336) - Uses the Combining Long Stroke Overlay character. This draws a full-width horizontal line through each character, producing the classic s̶t̶r̶i̶k̶e̶t̶h̶r̶o̶u̶g̶h̶ look. This is the most widely recognized and commonly used strikethrough effect.
  • Double Strikethrough (U+0335) - Uses the Combining Short Stroke Overlay character. This produces a shorter, thinner line through each character (s̵t̵r̵i̵k̵e̵t̵h̵r̵o̵u̵g̵h̵). The effect is more subtle and can be used when you want a less aggressive crossed-out appearance.

How Strikethrough Works in Unicode

The strikethrough effect is achieved using Unicode combining characters. These special characters are invisible on their own but attach to the preceding character and modify its visual rendering. When you type "H" followed by the combining long stroke overlay (U+0336), the rendering engine draws "H" with a line through it: H̶. Since combining characters are part of the Unicode standard (defined in the Combining Diacritical Marks block), they are supported on all modern devices, browsers, and operating systems.

Bold + Strikethrough Combo

We also offer a Bold + Strikethrough combination that first converts your text to Unicode Mathematical Bold characters and then applies the strikethrough combining character. This gives you bold crossed-out text like 𝗛̶𝗲̶𝗹̶𝗹̶𝗼̶ for maximum visual impact. This style is popular for emphasis in social media posts where you want the crossed-out text to really stand out.

The combining characters render correctly on iOS, Android, Windows, macOS, and Linux.
