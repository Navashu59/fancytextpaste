Square Text Generator - Text in Boxes Copy and Paste

The square text generator converts your text into Unicode characters where each letter is enclosed inside a square box. These "boxed letters" are part of the Unicode Enclosed Alphanumeric Supplement block and create a bold, structured look that is perfect for emphasis, headings, and creative text formatting. Unlike images or special apps, squared text is made of real Unicode characters that you can copy and paste anywhere.

Squared Character Variants

White Squared (🄷🄴🄻🄻🄾) places each letter inside an outlined square box. These have a clean, minimal look and display consistently across most platforms. Negative Squared (🅷🅴🅻🅻🅾) uses filled/solid square boxes with white letters inside, creating a high-contrast, bold appearance similar to keyboard keys or button labels. On some devices, negative squared letters may render with colored backgrounds as emoji.

How Squared Unicode Characters Work

Squared letters belong to the Unicode Enclosed Alphanumeric Supplement block (U+1F100 to U+1F1FF). Each character is a single Unicode code point -- the square and the letter are not separate elements but one unified character. This means they behave like any other text character: you can copy them, paste them, and they will display on any device or platform that supports these Unicode blocks. Only uppercase Latin letters (A-Z) have official squared variants.

Best Uses for Square Text

  • Emphasis and headings - Square text creates natural visual emphasis, making words stand out like labels or buttons
  • Creative formatting - Use boxed letters for section titles in social media posts and profiles
  • Aesthetic designs - Combine with other Unicode styles for unique text compositions
  • Gaming profiles - Squared letters create a bold, striking look for usernames and display names
  • Messaging - Add impact to key words in chats and comments

Square Text vs. Other Enclosed Styles

Square text is one of several "enclosed" Unicode styles. Bubble text (ⓐⓑⓒ) uses circles instead of squares for a rounder, softer look. Parenthesized text (⒜⒝⒞) wraps letters in parentheses for a subtler style. Each enclosed variant creates a distinct visual effect -- squares feel structured and bold, circles feel playful and soft, and parenthesized text feels minimal and understated.
