Unicode Support in Fortnite

Fortnite supports a wide range of Unicode characters in display names, including Mathematical Alphanumeric Symbols (U+1D400–U+1D7FF), Letterlike Symbols (U+2100–U+214F), and other supplemental Unicode blocks. The Epic Games font stack uses system defaults (Segoe UI on Windows, San Francisco on macOS) to render these characters across all platforms—PC, console, and mobile. Display names have a 20‑character limit (counting Unicode code‑points) and can be changed once every 14 days.

Epic Games Safety & Filtering

Epic Games employs a multi‑layer filtering system for in‑game chat, which blocks many non‑ASCII characters to prevent abuse. However, display names are exempt from these chat filters and allow full Unicode presentation. The platform’s automatic moderation still prohibits offensive content, but purely stylistic Unicode passes through. This separation ensures players can express themselves through stylish names while maintaining a safe chat environment.
