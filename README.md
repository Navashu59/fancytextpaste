# FancyTextPaste

Free fancy text generator for social media — Unicode fonts, symbols, kaomoji, and text styles.

## Features

- **30+ Text Generators** — Cursive, gothic, bold, italic, bubble, glitch, zalgo, vaporwave, and more
- **Platform-Specific Fonts** — Instagram, Twitter/X, Discord, TikTok, Facebook, LinkedIn, Snapchat
- **Symbol Collections** — Hearts, arrows, stars, cool symbols, Lenny faces, kaomoji, text art
- **Utility Tools** — Text repeater, invisible text, strikethrough, underline, wingdings translator

## Live Site

**[fancytextpaste.com](https://fancytextpaste.com)** — All tools are free. Just type, copy, and paste.
